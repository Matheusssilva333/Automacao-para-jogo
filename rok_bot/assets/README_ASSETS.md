# Image Assets for RoK Bot

The bot uses image recognition to interact with the game. Please place the following images in the `assets` directory:

## Buttons
- `buttons/search.png`: The magnifying glass icon to open the search menu.
- `buttons/search_confirm.png`: The yellow search button inside the search menu.
- `buttons/rally.png`: The 'Rally' button after clicking a fort.
- `buttons/rally_5min.png`: The 5-minute rally time option.
- `buttons/march_confirm.png`: The blue 'March' or 'Send' button.
- `buttons/alliance.png`: The Alliance menu button.
- `buttons/alliance_war.png`: The 'War' tab inside the Alliance menu.
- `buttons/join_rally.png`: The 'Join' button for an active rally.
- `buttons/gather.png`: The 'Gather' button for a resource node.
- `buttons/items.png`: The Items/Backpack icon.
- `buttons/use_item.png`: The 'Use' button for an item.
- `buttons/train.png`: The 'Train' button for troops.
- `buttons/train_confirm.png`: The final confirm button for training.
- `buttons/upgrade.png`: The 'Upgrade' button for buildings.
- `buttons/upgrade_confirm.png`: The final confirm button for building upgrade.
- `buttons/recall_gathering.png`: The recall button for a gathering march in the march list.
- `buttons/explore_fog.png`: The button to send a scout to a foggy area.
- `buttons/send_scout.png`: The button to confirm sending a scout.
- `buttons/explore_random.png`: The button to explore randomly from the scout camp.

## Tabs
- `tabs/barbarian_fort.png`: The Fort tab in the search menu.
- `tabs/rss_food.png`, `tabs/rss_wood.png`, etc.: Resource tabs in the search menu.
- `tabs/items_buffs.png`: The 'Buffs' tab in the items menu.

## UI Elements
- `ui/march_idle_icon.png`: Indicator that a march is idle.
- `ui/active_rally_icon.png`: Indicator in the Alliance button that a rally is active.
- `ui/scout_idle.png`: Indicator that scouts are idle.
- `ui/builder_idle.png`: Indicator that a building queue is free.
- `ui/upgrade_arrow.png`: The green arrow indicating a building can be upgraded.

## World Elements
- `world/barbarian_fort.png`: A barbarian fort on the map.
- `world/resource_node.png`: A resource node on the map.
- `world/tribal_village.png`: A tribal village on the map.
- `world/mysterious_cave.png`: A mysterious cave on the map.

## Buildings
- `buildings/barracks_icon.png`, `buildings/stables_icon.png`, etc.

## Indicators
- `red_screen_indicator.png`: Used to detect incoming attacks.
